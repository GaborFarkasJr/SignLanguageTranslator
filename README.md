# Pose Assisted Method for Isolated Sign Language Classification
This is a sign language recognition (more specifically classification) project that focuses on finding a flexible and intuitive method to classify isolated signs. The methodology and the set-up is designed to be easy-to-understand. For this, there is a detailed section to use and set up the files.
